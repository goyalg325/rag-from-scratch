# Advanced RAG Strategies

Advanced RAG systems often use query transformation, multi-query retrieval, re-ranking, and routing.

CRAG stands for Corrective RAG. It checks whether retrieval quality is strong enough before answering.
Self-RAG adds self evaluation so the system can critique its own evidence and decide whether to answer, broaden, or abstain.
Adaptive RAG uses routing rules to select direct answering, retrieval-heavy answering, or clarification.
